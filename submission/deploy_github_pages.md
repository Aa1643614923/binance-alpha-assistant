# GitHub Pages 快速部署

## 最简方式

1. 新建 GitHub 仓库，例如：`binance-alpha-assistant`
2. 把 `skills/binance-alpha-assistant` 整个目录上传到仓库根目录
3. 在 GitHub 仓库设置里开启 Pages
4. 选择从分支部署，指向包含 `demo/index.html` 的目录

## 建议的链接

- GitHub 仓库：`https://github.com/<yourname>/binance-alpha-assistant`
- 在线 Demo：`https://<yourname>.github.io/binance-alpha-assistant/demo/`
